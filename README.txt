== Description ==

Job Listings is a modern and powerful property listings plugin for your WordPress site.
It's created with built-in responsive design so that you can work with any theme.

= Features =

* Add, manage, and categorize job listings using the familiar WordPress UI.
* Work out of the box with any theme with responsive design.
* Powerful shortcodes for job listings.
* AJAX searchable & filterable job listings.

== Installation ==

1. Upload `job-listings` folder to the `/wp-content/plugins/` directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Job Listings -> Settings screen to configure the plugin.
